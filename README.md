# BakaDanmaku 模组
弹幕姬，不过是 Minecraft 模组版的；   
目前先做对 Bilibili 的兼容，后续可能会增加其他直播平台兼容。  
基于 3TUSK 的模组修改：<https://github.com/3TUSK/BiliLiveMonitor>
